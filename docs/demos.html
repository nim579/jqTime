jqTime 2.3

— это jQuery плагин, выводящий время на экран. Плагин содержит в себе множество гибких настроек.

Тестирование и сборка

В проекте используется тестированиес помощью qUnit. Пока весь проект не покрыт тестами, но тесты дописываются постоянно.

Разработка проекта ведетcя с методологией Continuous Integration, автоматизированное тестирование работает на Travis CI.

Для сборки проекта используется Grunt.js.
